# 📦 Controle de Estoque

Sistema de controle de estoque desenvolvido em **Java**, aplicando os conceitos de **Programação Orientada a Objetos (POO)**, modelagem **UML** e o padrão de projeto **Builder**.

---

## 📖 Sobre o projeto

O **Controle de Estoque** é um sistema desenvolvido para gerenciar produtos, permitindo cadastrar itens, controlar entradas e saídas de mercadorias, buscar produtos, consultar informações do estoque e gerar relatórios.

Este projeto foi desenvolvido como atividade da disciplina de **Programação Orientada a Objetos**, com o objetivo de aplicar conceitos de orientação a objetos, modelagem UML e o padrão de projeto **Builder**.

---

## ✨ Funcionalidades

- ✅ Cadastro de produtos eletrônicos
- ✅ Cadastro de produtos perecíveis
- ✅ Listagem de produtos
- ✅ Busca de produtos por código
- ✅ Entrada de estoque
- ✅ Saída de estoque
- ✅ Remoção de produtos
- ✅ Listagem de produtos com estoque baixo
- ✅ Cálculo do valor total do estoque
- ✅ Validação de dados informados pelo usuário
